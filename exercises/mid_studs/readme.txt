Создайте программу на C++, которая будет считывать данные о студентах из файла и 
выводить информацию о студентах, имеющих оценки выше среднего балла.
В программе должна быть использована структура для хранения информации о студенте.

Структура должна содержать следующие поля:
- ФИО студента;
- оценки по предметам.

Функция для вычисления среднего балла должна принимать массив структур в качестве 
аргумента и возвращать среднее арифметическое оценок всех студентов.
Программа должна считывать данные о студентах из файла, создавать динамический массив 
структур, вычислять средний балл и выводить информацию о студентах, имеющих оценки выше среднего.
