Техническое задание для проекта "Система управления автопарком"
Описание проекта
Необходимо разработать систему управления автопарком, которая позволит управлять автомобилями и водителями. 
Система должна включать функционал для добавления, удаления, поиска и отображения информации об автомобилях и водителях, 
а также назначения автомобилей водителям и снятия автомобилей с водителей.

Требования к системе
Классы и объекты:

1.1. Класс Car:

Поля: ID, brand, model, year, mileage.
Методы: геттеры и сеттеры для всех полей.
1.2. Класс Driver:

Поля: ID, firstName, lastName, assignedCars (список назначенных автомобилей).
Методы: геттеры и сеттеры для всех полей, методы для добавления и удаления автомобилей из списка назначенных.
Наследование:

2.1. Создать суперкласс FleetItem, который будет содержать общие свойства и методы для классов Car и Driver.

Поля: ID.
Методы: геттеры и сеттеры для поля ID.
2.2. Классы Car и Driver должны наследовать FleetItem.

Инкапсуляция:

Защитить данные в классах с использованием модификаторов доступа (private или protected).
Обеспечить доступ к полям классов через методы (геттеры и сеттеры).
Полиморфизм:

4.1. Создать интерфейс IFleetManagement, который будет содержать методы для управления автопарком:

addCar(Car car).
removeCar(int carID).
findCarByBrand(String brand).
findCarByModel(String model).
findCarByYear(int year).
addDriver(Driver driver).
removeDriver(int driverID).
findDriverByFirstName(String firstName).
findDriverByLastName(String lastName).
assignCarToDriver(int carID, int driverID).
unassignCarFromDriver(int carID, int driverID).
displayAllCars().
displayAllDrivers().
4.2. Реализовать этот интерфейс в классе Fleet.

Абстракция:

Скрыть внутреннюю реализацию управления автомобилями и водителями за интерфейсом IFleetManagement.
Функциональность:

6.1. Добавление автомобилей и водителей.

6.2. Удаление автомобилей и водителей.

6.3. Поиск автомобилей:

По марке.
По модели.
По году выпуска.
6.4. Поиск водителей:

По имени.
По фамилии.
6.5. Назначение автомобилей водителям.

6.6. Снятие автомобилей с водителей.

6.7. Отображение списка всех автомобилей и водителей.
